# Desmos Dark Mode Extension (Chrome)

Automatically enables Reverse Contrast (dark mode) on Desmos calculator pages.

## Installation

1. Unzip the file
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right)
4. Click "Load unpacked"
5. Select this folder
6. The extension is now active

## Supported Pages
- /calculator
- /3d
- /geometry
- /scientific
- /fourfunction
- /matrix
